Installation Guide

Official instructions are available here:

  1. Visit the Lua binaries webpage and locate the section appropriate to your operating system.
  2. Download the executable file. For better linkage to C modules, download the DLL version too!
  3. Extract all file contents to a single folder. Set up an operating system environment path to this folder.
  4. Call the main executable from the command line to check success. (It's recommended to rename all executables to omit version numbers. Doing this changes the command to "lua".)
